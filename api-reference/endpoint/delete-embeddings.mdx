### Sample Request
```bash
curl --request DELETE \
  --url https://api.usecortex.ai/embeddings/delete \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
  "sub_tenant_id": ""
}'
  ```
