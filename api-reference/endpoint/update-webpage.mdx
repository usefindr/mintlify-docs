### Sample Request
<TryIt />
```bash
curl --request PATCH \
  --url 'https://api.usecortex.ai/upload/update_webpage?web_url=https%3A%2F%2Fwww.usecortex.ai%2F&source_id=CortexDoc1234&tenant_id=tenant_1234&sub_tenant_id=sub_tenant_4567' \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/x-www-form-urlencoded'
  ```

If the `source_id` does not exist, a new entry with that source ID will be created. This endpoint essentially performs an upsert (update or insert) operation.
