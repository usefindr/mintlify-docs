---
title: 'Upload Text'
openapi: 'POST /upload/upload_text'
---

### Functionality
- Validates that content is provided
- Generates a unique ID for the uploaded content
- Creates secure access tokens for processing
- Triggers the ingestion pipeline for content processing and indexing
- Returns the generated ID for future reference

> **Note**: If `file_id` is not provided in the document metadata, the system will generate an ID automatically.

## Sample Request

```bash
curl --location 'https://api.usecortex.ai/upload/upload_text?tenant_id={TENANT_ID}&sub_tenant_id={SUB_TENANT_ID}' \
  --header 'Authorization: Bearer {API_KEY}' \
  --header 'accept: application/json' \
  --header 'Content-Type: application/json' \
  --data '{
    "content": "This is sample markdown content that will be processed and indexed.",
    "tenant_metadata": {
      "department": "Finance",
      "compliance_tag": "GDPR"
    },
    "document_metadata": {
      "title": "Q1 Report.pdf",
      "author": "Alice Smith",
      "file_id": "custom_text_123"
    }
  }'
```
