title: 'Delete User Memory'
openapi: 'DELETE /user_memory/delete_user_memory'
---

### Functionality
- Accepts user memory ID in the request
- Calls the delete function with the memory ID and user context
- Returns confirmation of deletion success/failure

### Headers
- `Authorization`: Bearer token (your API key)
- `accept`: application/json
- `Content-Type`: application/json
