---
title: 'Delete User Memory'
openapi: 'DELETE /user_memory/delete_user_memory'
---

### Functionality
- Accepts user memory ID in the request
- Calls the delete function with the memory ID and user context
- Returns confirmation of deletion success/failure

## Sample Request

```bash
curl --location 'https://api.usecortex.ai/user_memory/delete_user_memory?tenant_id={TENANT_ID}&sub_tenant_id={SUB_TENANT_ID}&memory_id={MEMORY_ID}' \
  --header 'Authorization: Bearer {API_KEY}' \
  --header 'accept: application/json' \
  --header 'Content-Type: application/json'
```

### Headers
- `Authorization`: Bearer token (your API key)
- `accept`: application/json
- `Content-Type`: application/json
