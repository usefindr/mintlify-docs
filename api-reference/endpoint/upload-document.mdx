Uploads a document (e.g., PDF, DOCX) for processing and indexing.

## Sample Request

```bash
curl -X POST https://api.usecortex.ai/upload/upload_document \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@document.pdf" \
  -F "tenant_id=tenant_456" \
  -F "custom_metadata={\"author\":\"John Doe\",\"department\":\"Sales\",\"tags\":[\"contract\",\"Q1\"]}"
```
